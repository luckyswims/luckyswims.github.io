Meal Planning App

Ruby on Rails, Mongoose, MongoDB, Bootstrap, Handlebars

A application for creating meal plans for the week. The website currently allows a user to create a meal plan containing a list of seven meals, stored as strings. The user can then see all of their meal plans, edit their meal plans, delete their meal plans, and search for a specific meal plan.

The end goal for this project is for the user to be able to create a library of meals, with each meal having ingredient lists and instructions. The user would then be able to build a meal plan that references meals from their library. The application would then put together a list of ingredients and quantities that they need to have in order to make their meals.
